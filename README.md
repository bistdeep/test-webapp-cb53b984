# Test Project

## Overview
This is a test project for a static web application that displays data from a JSON file and a CSV file, with additional features such as filtering, theme toggling, and responsive design.

## Features
- Displays data from a JSON file.
- Displays data from a CSV file in a table format.
- Filtering capability for the CSV data table.
- Dark/light theme toggle.
- Details view for JSON data items when clicked.
- Fully responsive design for mobile devices.

## Setup Instructions
1. Clone the repository or download the files.
2. Open `index.html` in a web browser.

## Usage
- Use the filter input to search for products in the CSV data table.
- Click on a product row to view more details about the product.
- Toggle between dark and light themes using the button provided.

## Code Explanation
- The HTML structure includes a header, a filter input, a table for CSV data, and a details section.
- CSS styles are included inline for simplicity and responsiveness.
- JavaScript functions handle theme toggling, filtering, and displaying details for each product.
